# Chrome Web Store image assets

Upload the following files in the Chrome Web Store Developer Dashboard.

## Required assets

- `store-icon-128.png` — extension/store icon, 128×128 PNG.
- `screenshot-01-score-explanation-1280x800.png` — real extension score and evidence popover on a synthetic profile.
- `screenshot-02-complete-scan-1280x800.png` — real completed-scan result.
- `screenshot-03-dark-mode-1280x800.png` — real extension score and evidence popover in dark mode.
- `screenshot-04-popup-1280x800.png` — real extension popup composited onto branded artwork.
- `screenshot-05-complete-scan-dark-1280x800.png` — real completed-scan result in dark mode.
- `promo-small-440x280.png` — required small promotional tile.

## Optional assets

- `promo-marquee-1400x560.png` — optional marquee promotional image.

All screenshots and promotional tiles are RGB PNG files without an alpha channel. The screenshots use synthetic profile information and UI captured from the current Chromium build. The popup composition preserves the captured popup without redrawing its controls.
